# tistory-useful-scripts

> 티스토리에서 여러모로 불편한 부분들을 해결할 수 있는 스크립트를 아카이빙

### 적용 방법:
1. 티스토리 에디터의 우측 상단에서 모드를 `HTML` 모드로 변경
2. 적용하려는 스크립트를 복사
3. 에디터 첫 번째 줄에 적용

<br>

### Script 목록:
- `darkmode_color_fix.js`: 다크모드에서 글자색 전환 안 되는 버그 / [출처](https://dev-mintcat.tistory.com/12)
